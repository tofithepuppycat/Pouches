This is a modification for Minecraft, version 1.20.1

Pouches are little containers that you wear in a new Curio slot! By default, each player has 3 pouch Curio slots, which can be further increased using the new enchantment - Pouched. Different tiers of pouches can be equipped, that differ in the amount of item slots offered (2, 3, 4 or 6 slots). By default, holding the Open Pouch Wheel key (bound to [C]) opens a selection wheel and releasing the key applies the selected slot. A client config option (`input.togglePouchWheelMode` in `pouches-client.toml`) can switch this to toggle mode: press once to open the wheel and press again to apply the selected slot. By sneaking, the player cycles through their equipped pouches. Pouches can also be stored inside of other pouches (please don't exploit this), so one can carry multiple sets of items in one slot, kind of like a ender chest full of shulker boxes (though on a much smaller scale).

Curios API is a required dependency!
