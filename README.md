Pouches are little containers that you wear in a new Curio slot! By default, each player has 3 pouch Curio slots, which can be further increased using the new enchantment - Pouched. Different tiers of pouches can be equipped, that differ in the amount of item slots offered (2, 3, 4 or 6 slots). When held, the Open Pouch Wheel key (Bound to [C] by default) opens a selection wheel, where you can choose a slot to put items in. By sneaking, the player cycles through their equipped pouches. Pouches can also be stored inside of other pouches (please don't exploit this), so one can carry multiple sets of items in one slot, kind of like a ender chest full of shulker boxes (though on a much smaller scale).

Curios API is a required dependency!
